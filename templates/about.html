About SignSecure

Your Signature Defines You. We Defend It.

What We Do

SignSecure is a signature verification system that helps detect forged signatures and ensures the authenticity of handwritten signatures. Here's how it works:

  1. Upload or Draw a Signature: Users can upload a signature image or draw one on a canvas.
  2. Compare Signatures: The system compares the uploaded or drawn signature with a reference signature stored in the database.
  3. Get Results: The system calculates a similarity score and determines whether the signatures match.

Tools We Use

SignSecure is built using state-of-the-art technologies and tools:

  • Siamese Neural Networks: For signature similarity detection.
  • PyTorch: For building and training deep learning models.
  • Flask: For building the web application.
  • Bootstrap: For designing a responsive and user-friendly interface.
